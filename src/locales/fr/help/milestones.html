Liste des versions

Nom
Le nom de la version. Le lien donne accès au détail de la version.
Date
La date de publication prévue/réelle de la version.
Tickets
Le nombre de tickets associés à cette version :
  • Le nombre total de tickets.
  • Le nombre de tickets actifs (nouveaux ou acceptés).
  • Le nombre tickets fermés.
  • Le nombre tickets "déchets" (rejettés ou supprimés).

A chaque fois, le lien donne accès à la liste des tickets correspondants.

État
L'état de la version :
  • à venir : la version est en préparation.
  • publié : la version a été publiée.

Filtres de la liste des versions

État
Filtre en fonction de l'état de la version :
  • à venir : la version est en préparation.
  • publié : la version a été publiée.

Editer une version

Nom
Inscrivez le nom/numéro de la version. Ce champ est obligatoire.
Description
La description de la version. Ce champ est obligatoire.
État
Permet de choisir l'état de la version après enregistrement :
  • à venir : la version est en préparation.
  • publié : la version a été publiée.
Prévue pour le/Publié le
Permet de modifier la date et l'heure de publication de la version. Ce champ est obligatoire.
URL spécifique
Ce champ permet de choisir une URL pour la version autre que celle par défaut après avoir cliqué sur le petit verrou placé à sa droite. Si vous essayez d'utiliser une URL déjà existante, celle-ci se verra incrémentée d'un chiffre. Ce champ est obligatoire.
Pièces jointes
Les pièces jointes sont tous les media attachés à la version (sources, binaires, documentation, etc.). Le lien d'ajout d'une pièce jointe permet de choisir un fichier à ajouter à la version. Vous pouvez également attacher des médias directement au projet.
